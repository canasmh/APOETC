filter(bandpass, Johnson=True, SDSS=False)[source]¶

Method that returns the transmission of specified filter.

Parameters

  • bandpass (str) – The bandpass of the filter used (i.e., ‘U’,’B’,’V’,’R’, or ‘I’).

  • Johnson (bool, optional) – If true, then the bandpass is referring to the Johnson-Cousin filters. Defaults to True

  • SDSS (bool, optional) – If true, then the bandpass is referring to the Johnson-Cousin filters. Defaults to False

Returns

The transmission of the filter interpolated over the bandpass. Also sets a filter_range attribute (Angstroms).
